The technology we are using on these pages is incredibly
complex, but to think about it, there are only three main
programs we use.

HTML is where the main coding takes place. The HTML is where
all the main things like text, links, and columns are placed
and configured. If you want a paragraph describing HTML in your
website, that is where you'd want to go.

CSS is basically the makeup and the accessories of the website.
In this program, you are able to add background images, change
text color, and create borders and margins.

Bootstrap is the latest and newest of the three that iv'e used
so far. In this program, you can organize and customize the website.
You can create borders and customize said border. you can create
rows and colomns to organize your website with.
